# WP Block Studio – Plugin Distribution Repository

This repository is used **exclusively for distributing release builds** of the WP Block Studio WordPress plugin.

## Purpose

- Hosts versioned release artifacts (ZIP files)
- Serves as the update source for licensed installations
- Separates development source code from distribution assets

No development source code is maintained in this repository.

## Releases

All official plugin builds are published via **GitHub Releases** and are intended to be consumed by WordPress’s automatic update system.

Release archives contain:
- Compiled JavaScript and CSS assets
- Production-ready PHP files
- No development source, build tooling, or raw modules

Use, copying, modification, or redistribution of this software is **not permitted** without explicit permission from the author, except as allowed under an active license agreement.

Access to release assets is provided solely for the purpose of installing and updating licensed copies of the plugin.

## Access

This repository is private.

Release assets are accessible only to authenticated clients as part of a licensed update flow. Links to release assets are not intended for public distribution.
